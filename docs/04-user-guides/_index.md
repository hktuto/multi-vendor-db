# User Guides

Client-facing documentation with step-by-step guides and screenshots.

## Available Guides

```dataview
LIST
FROM "04-user-guides"
WHERE file.name != "_index"
SORT file.name ASC
```

## Guide Template

Each guide should include:
1. **Overview** - What this feature does
2. **Prerequisites** - What's needed before starting
3. **Step-by-Step** - Numbered instructions with screenshots
4. **Troubleshooting** - Common issues and solutions
5. **FAQ** - Frequently asked questions

---

*Create guides using Chrome MCP screenshots after feature testing is complete.*
