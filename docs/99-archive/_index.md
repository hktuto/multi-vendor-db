# Archive

Completed and cancelled features moved here for historical reference.

## Completed Features

```dataview
LIST
FROM "99-archive"
WHERE status = "finish"
SORT completed DESC
```

## Cancelled Features

```dataview
LIST
FROM "99-archive"
WHERE status = "cancel"
SORT completed DESC
```

---

*Features are moved here from `01-features/` when marked as `finish` or `cancel`.*
